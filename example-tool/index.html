Example Tool

A demonstration of the folder-based routing structure

How it works

This is an example tool page. Each tool lives in its own folder, making the URL structure clean and organized.

Creating a New Tool

  1. Create a new folder in the root directory (e.g., my-tool/)
  2. Add an index.html file in that folder
  3. Link the shared styles.css using ../styles.css
  4. Add a card for your tool in the main index.html

URL Structure

Your tool will be accessible at: https://yourusername.github.io/tools/folder-name/
